# Celestia Crest Roofing Contractors

Professional roofing services website serving 7 South Florida communities: Coral Gables, Dania Beach, Homestead, North Miami Beach, Opa-locka, Pembroke Pines, and Plantation.

**Live site:** [celestiacrestroofingcontractors.com](https://celestiacrestroofingcontractors.com)

## Structure

```
index.html                                          ← Main homepage

```

## Tech Stack

- Static HTML + CSS (no build tools, no frameworks)
- Google Fonts: Cormorant, Plus Jakarta Sans, Manrope
- Font Awesome 6.5.1 (local mirror)
- Vanilla JavaScript for interactive components (before/after sliders, mobile menu)
- Schema.org structured data (Organization + LocalBusiness) for SEO
- Embedded Google Maps per location

## Local Development

No install required. Open `index.html` directly or serve with any static server:

```bash
npx serve .
```
